# fanc-terraform


## 概要
 [PitaScho](https://pitascho.com/)の管理サービス[fanc](https://fancapp.com/login)のAWS環境をterraformで管理するリポジトリ

## 使用サービス概要

- ECS(Fargate)
- CloudWatch
- VPC
- ECR
- S3
- ACM
- CloudFront
- IAM
